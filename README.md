# Sistema de Gestión de Bar

## Descripción

Este proyecto es un sistema web desarrollado en **PHP** utilizando la arquitectura **Modelo-Vista-Controlador (MVC)**. Fue creado con el objetivo de fortalecer mis conocimientos en el desarrollo de aplicaciones web aplicando buenas prácticas de programación y una estructura organizada del código.

El sistema está enfocado en la administración de un **bar ficticio**, permitiendo gestionar de forma eficiente las principales operaciones del negocio mediante una interfaz intuitiva y funcionalidades completas.

## Características

* Inicio de sesión (Login).
* Panel de administración (Dashboard).
* Operaciones CRUD (Crear, Leer, Actualizar y Eliminar).
* Gestión de productos.
* Gestión de categorías.
* Gestión de usuarios.
* Gestión de clientes.
* Registro y administración de ventas.
* Generación de reportes en formato PDF.
* Reportes de ventas.
* Validación de formularios.
* Arquitectura basada en el patrón MVC.

## Tecnologías utilizadas

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript
* Bootstrap
* Arquitectura MVC

## Objetivo

El propósito de este proyecto es demostrar la implementación de una aplicación web completa utilizando PHP y el patrón MVC, desarrollando funcionalidades propias de un sistema de gestión empresarial y aplicando buenas prácticas de desarrollo.
